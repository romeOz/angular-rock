Client side for [Rock Framework](https://github.com/romeOz/rock) using the AngularJS
=================

Installation
-------------------

From the Command Line:

`bower install angular-rock`

In your bower.json:

```json
{
    "dependencies": {
        "angular-rock": "*"
    }
}
```

Requirements
-------------------
 * **AngularJS 1.3**
